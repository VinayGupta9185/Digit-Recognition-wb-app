About Project

This project is on Handwritten digit recognition between 0 to 9.
In this project ,the model used is deep learning model using CNN , Tensorflow and Keras Library.
After training and evaluating on MNIST dataset the model predict the digits between 0 to 9 with the accuracy of 98.94%.
Node.js is mainly used to build this app whereas the languages used is Javascript , Python .
